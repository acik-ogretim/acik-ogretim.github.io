# Güvenlik Politikası

Açık Öğretim Portal projesinin güvenliğine önem veriyoruz.

## Desteklenen Sürümler

Şu anda projenin sadece en son sürümü (`main` branch) güvenlik güncellemeleri için desteklenmektedir.

| Sürüm | Destekleniyor mu? |
| :-: | :-: |
| main | ✅ |
| Eski sürümler | ❌ |

## Güvenlik Açığı Bildirme

Eğer projede bir güvenlik açığı, hassas veri sızıntısı veya kötüye kullanılabilir bir hata keşfederseniz:

1. Lütfen bu durumu **açık (public) bir Issue olarak bildirmeyin.**
2. Bunun yerine, GitHub üzerindeki [Security Advisory](https://github.com/acik-ogretim/acik-ogretim.github.io/security/advisories) özelliğini kullanarak veya varsa proje sahiplerine doğrudan ulaşarak bildirimde bulunun.
3. Bildiriminiz incelenecek ve en kısa sürede dönüş yapılacaktır.
4. Sorun çözülene kadar detayı gizli tutmanızı rica ederiz.

Projemizi daha güvenli hale getirmemize yardımcı olduğunuz için teşekkürler!
